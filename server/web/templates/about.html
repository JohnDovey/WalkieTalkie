What it is

WalkieTalkie is a LAN push-to-talk mesh. Hold Talk on any client and other devices on the same Wi‑Fi hear you live — no accounts, no manual pairing. This computer is a Base Station: it joins the same mesh as Android, iPhone, Wear OS, and Apple Watch (via iPhone), and hosts the web dashboard you are using now.

How discovery works

Peers find each other with mDNS over Wi‑Fi. Off the LAN, nearby phones and watches can still appear over Bluetooth LE as presence-only (id and name, no live audio until they are back on Wi‑Fi together).

The Base Station

Besides Talk, this Base Station is the mesh hub for the device list, map, settings, and store-and-forward voice notes / private channels for mobile clients. Group Hold-to-talk works peer-to-peer without a Base Station; voice messages and private chats need one on the LAN. Open the dashboard link above anytime — it is this UI’s own address on the network.
